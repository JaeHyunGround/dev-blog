# MDX에서 이미지 삽입하기

블로그 포스트를 작성할 때 이미지는 필수입니다. MDX에서 이미지를 삽입하는 방법을 알아보겠습니다.

## 기본 마크다운 문법

가장 간단한 방법은 마크다운 이미지 문법을 사용하는 것입니다:

```markdown
![이미지 설명](/images/posts/image-example/screenshot.png)
```

아래는 실제 이미지 예시입니다:

![예시 스크린샷](/images/posts/image-example/screenshot.png)

## 이미지 폴더 구조

이미지는 `public/images/` 폴더에 저장합니다:

```
public/
└── images/
    └── posts/
        └── image-example/      ← 포스트별 폴더
            ├── thumbnail.png   ← 썸네일
            └── screenshot.png  ← 본문 이미지
```

## 이미지에 링크 걸기

이미지를 클릭하면 다른 페이지로 이동하게 할 수 있습니다:

```markdown
[![이미지 설명](/images/posts/image-example/logo.png)](https://nextjs.org)
```

## 정리

| 방법 | 문법 | 용도 |
|------|------|------|
| 기본 | `![alt](src)` | 일반 이미지 |
| 링크 | `[![alt](src)](url)` | 클릭 가능한 이미지 |

이미지 파일은 `public/images/posts/포스트명/` 폴더에 정리해서 관리하면 깔끔합니다.
